# Puppeteer Boilerplate
    
Puppeteer end-to-end test automation boilerplate, Using Commonjs, Mocha, Chai and Assert.
    
## Getting Started
    	1. git clone git@github.com:e2e-boilerplate/puppeteer-commonjs-mocha-chai-assert.git
    	2. cd puppeteer-commonjs-mocha-chai-assert
    	3. npm install
    	4. npm run test
